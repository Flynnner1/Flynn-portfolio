How the Kill Counter Works
The kill counter in Arcana Rift keeps track of how many enemies you have defeated during your adventure.

  • The counter updates in real-time as you defeat slimes, skeletons, zombies, and other enemies.
  • When you press K you instantly see your progress of how many you've killed of a certain enemies.
  • The kill counter can be used for personal high scores or to showcase how many rare versions of enemies you've killed.
How the Code Works
Example: Saving and Loading the Kill Counter (Unity C#)
This snippet demonstrates how the kill counter can be saved and loaded using Unity's PlayerPrefs system. Replace it with your real code or expand as needed for your project! 
// When you kill an enemy
killCount++;
PlayerPrefs.SetInt("KillCount", killCount);
PlayerPrefs.Save();

// To load the kill count when the game starts
void Start() {
    LoadCounts();
}
