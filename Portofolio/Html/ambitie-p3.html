Arcana Rift

A showcase of my skills and work, built with Unity, C# and pixelart.

Overview

This portfolio project highlights my game development journey at Sint-Lucas. It features a game I made myself for the school showcase, demonstrating my ability to create a very technical game and showcase different skills.

Features

  • A working inventory
  • A quest giver
  • A lot of variations of enemies
  • A lot of different attacks/spells
  • A leveling system
  • A HP system
  • A place where enemies spawn with a certain chance to be more rare
  • A kill counter

Technologies Used

  • C#
  • Unity
  • Pixelart

What I Learned

  • Building an inventory
  • Making items to put into that inventory
  • Making it so that the items can move around your inventory
  • Making a hotbar and connecting it with your keys
