Pong: Learning the Fundamentals

A foundational project focused on understanding the core game loop and the basics of game development.

Overview

This project was my first step into game development, where I recreated the classic game of Pong. The goal was not to build a complex game, but to gain a deep understanding of the fundamental principles that make a game work, especially the concept of a game loop.

What I Learned: The Core Basics

  • The Game Loop: I learned how a game constantly runs through a loop to update its state and render the results. This includes the `Update()` and `FixedUpdate()` cycles for logic and physics, and how the camera renders frames.
  • Player Input: I implemented code to listen for keyboard input to move the player paddles up and down, connecting the player's actions to the game world.
  • Basic Physics & Collision: I wrote logic to handle the ball's movement and to detect when it collided with the paddles or the top/bottom walls, making it bounce realistically.
  • Scoring and Game State: I built a simple system to track player scores and reset the ball's position after a point was scored, managing the basic state of the game.
