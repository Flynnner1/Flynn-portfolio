Fake News Project: GIN

A parody of IGN gaming news for web development.

Overview

The Fake News project, GIN (Gaming Inaccurate News), is a direct and as accrurate webstite and inspired by gaming media IGN. this project was very fun to make because we had something to work off and copy it.

Project Goals

  • showcasing HTML skills
  • Practice and learning basic skills for website building
  • Experiment with coping IGN.

How GIN Works

GIN features totally fake, but realistic news about fake news articals, such as Minecraft 2 being released. The goal is to entertain visitors while demonstrating web layout, CSS styling, and content structuring skills.

Technologies Used

  • HTML for structure
  • CSS for responsive, modern styling

What I Learned

  • Building single-page parody websites
  • Styling with CSS for almost the same website like IGN
